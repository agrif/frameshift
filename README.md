# frameshift

[![crates.io](https://img.shields.io/crates/v/frameshift.svg)](https://crates.io/crates/frameshift)
[![docs.rs](https://docs.rs/frameshift/badge.svg)](https://docs.rs/frameshift)

frames, in both space and time.
